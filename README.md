Fungsi utama program ini adalah sebagai kalkulator otomatis untuk menentukan total biaya pengiriman paket.
Program ini mengambil aturan bisnis yang Anda berikan (biaya berdasarkan berat, jarak, dan volume) dan mengubahnya menjadi aplikasi interaktif yang dapat digunakan siapa saja untuk menghitung biaya secara akurat.

Cara kerja program ini dapat dibagi menjadi tiga langkah utama:
Input Data (Mengumpulkan Informasi)
1.Program akan meminta pengguna memasukkan semua data yang diperlukan melalui konsol.
2.Output Hasil (Menampilkan Total)
3.Program menjumlahkan kedua hasil perhitungan tersebut (Biaya Jarak/Berat + Biaya Volume).
Hasil akhirnya (Total Biaya) ditampilkan di layar pengguna dengan format mata uang yang rapi.
